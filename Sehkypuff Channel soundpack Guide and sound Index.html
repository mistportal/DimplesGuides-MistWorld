Introduction

The following sound pack has been compiled from other packs used in audio games or MUDs to enhance accessibility. The default files in Mist World are too quiet and a little bland. This pack features recognizable sounds that you've likely heard before. Sehkef has curated this collection, referred to as the Sehkefypuff Channel Soundpack. The download button for the soundpack is at the bottom of the page, just press B!

Installation Instructions

  1. If you use English as your main language in the game, do not rename the folder, simply copy the English folder and paste it into the sound folder of your Mist World directory and restart the game.
  2. If you use another language like Spanish, rename the folder accordingly.
  3. Alternatively, enter the English folder, select all the sounds, and copy them.
  4. Navigate to your Mist World directory and enter the "sounds" folder.
  5. Inside the "sounds" folder, create a new folder named after the language you use in the game.
  6. If you already use custom sounds and have completed this step before, simply enter this folder and paste the sounds.
  7. Restart Mist World to apply the changes.

Now you'll have clear stereo sounds to distinguish all your channels. Enjoy the enhanced experience!

Note: All sounds are in .ogg format to reduce file size. There is one sound file in .wav format (team sound) that only works in .wav due to technical reasons.

Channel Sound Index

  • 1328 = Global
  • 1329 = Map
  • 1330 = Nearby
  • 1331 = Gang
  • 1334 = Team
  • 1335 = International
  • 1336 = Private Channel
  • 1025 = Private Message

This information is also available in the SID.ini file located in your config folder within the Mist World directory, but as they were likely translated from Chinese, they are not labelled very well. Feel free to replace sounds according to these IDs if desired.
